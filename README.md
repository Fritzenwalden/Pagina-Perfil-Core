# Pagina-Perfil-Core
Estructura básica de una página de perfil profesional que incluye en html y css

HTML Incluye:
Encabezado con navegación.
Sección de información del usuario con imagen, descripción y botón de edición (Mock-up).
Panel lateral con solicitudes de conexión y lista de contactos.
Estructura en dos columnas adaptable (grid).

CSS:
Aplica un diseño moderno con tarjetas, sombras suaves, tipografía legible y colores planos.
El layout es responsivo gracias a media queries que ajustan la disposición a pantallas pequeñas (se transforma en una sola columna bajo los 900 px).
